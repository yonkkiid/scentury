# Scentury - Конструктор парфюмерии

Многостраничный сайт для создания индивидуальных ароматов с интерактивным конструктором.

## Описание проекта

Scentury - это современный веб-сайт, позволяющий пользователям создавать уникальные ароматы с помощью интерактивного конструктора. Сайт включает в себя каталог готовых ароматов, блог о парфюмерии, информацию о компании и форму для заказа.

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP 7.4+
- **Стили**: CSS Grid, Flexbox, адаптивная верстка
- **Интерактивность**: Vanilla JavaScript

## Функционал

### Главная страница (index.php)
- Привлекательный баннер с призывом к действию
- Блок "Как это работает" с пошаговой инструкцией
- Галерея популярных ароматов
- Отзывы клиентов
- Адаптивная навигация

### Конструктор ароматов (constructor.php)
- **Интерактивный выбор нот** по трем категориям:
  - Верхние ноты: Бергамот, Лимон, Лайм, Мандарин
  - Сердечные ноты: Роза, Жасмин, Лаванда, Иланг-иланг
  - Базовые ноты: Ваниль, Сандал, Пачули, Амбра
- **Предпросмотр аромата** с отображением выбранных нот
- **Поле для названия** создаваемого аромата
- **Кнопка "Отправить заявку"** с переносом данных в форму контактов

### Каталог (catalog.php)
- Сетка карточек с готовыми ароматами
- **Фильтры по цене и типу** аромата
- Детальная информация о каждом аромате
- Кнопки заказа

### О компании (about.php)
- Информация о миссии и истории компании
- Статистика достижений
- Команда специалистов
- Ценности компании

### Блог (blog.php)
- Статьи о парфюмерии
- Категории: История, Образование, Ингредиенты, Советы, Стиль, Мастер-класс
- Пагинация
- Адаптивная сетка статей

### Контакты (contacts.php)
- **Форма заказа** с полями:
  - Имя, телефон, email (обязательные)
  - Название аромата
  - Выбранные ноты
  - Дополнительные пожелания
- Контактная информация
- Карта проезда
- Социальные сети

## Установка и запуск

1. **Клонируйте проект** в папку веб-сервера
2. **Убедитесь, что PHP включен** и работает
3. **Откройте сайт** в браузере по адресу `http://localhost/scentury/`

### Требования
- PHP 7.4 или выше
- Веб-сервер (Apache, Nginx)
- Современный браузер с поддержкой ES6+

## PHP функционал

### Обработка форм (process-form.php)
- Валидация входящих данных
- Защита от спама
- Сохранение заказов в файл `orders.txt`
- Отправка email уведомлений (настраивается)

### Безопасность
- Валидация всех входящих данных
- Защита от XSS атак
- Проверка обязательных полей

## Настройка email

Для отправки email уведомлений раскомментируйте строку в `process-form.php`:

```php
mail($to, $subject, $email_message, $headers);
```

И настройте SMTP сервер в соответствии с вашими требованиями.

## Файлы данных

Заказы сохраняются в файл `orders.txt` в формате JSON. Каждая строка содержит один заказ.
